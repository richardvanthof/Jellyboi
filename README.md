# Jellyboi
Arduino hardware test project for the Jellyfish creature.

## Requirements
- Arduino UNO (or equivalent) + Hardware
- Arduino IDE

## Getting started
1. Set up the hardware using the following [schematic](https://www.tinkercad.com/things/eZMfBvmO5Tu/editel?returnTo=%2Fdashboard&sharecode=OJPrvETBBBBATBOS4KXb_3DIvOjOb9DPG6LTnJ8mWJM).
2. Clone the repository.
3. Upload the code from `src/jellyboi.ino.` to your Arduino.
